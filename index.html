HTML 5

a

HTML 5 es la última versión de HTML. El término representa dos conceptos diferentes: Se trata de una nueva versión con nuevos elementos, atributos y comportamientos, y un conjunto más amplio de tecnologías que permite a los sitios web y a las aplicaciones ser más diversas y de gran alcance. Este conjunto se conoce a veces como HTML5 y amigos y a menudo se abrevia como HTML5.

[21/04/2024. Andrés Fernández Cano]

CSS 3

a

CSS 3 es el último estándar de CSS hasta la fecha. Introdujo muchas características nuevas y mejoradas, como selectores avanzados, animaciones, sombras, bordes redondeados, transiciones, flexbox, grid, entre otros. Estas características permiten a los desarrolladores crear diseños más sofisticados y atractivos para sus sitios web. Además, CSS 3 también ofrece mejoras en la compatibilidad con diferentes navegadores, lo que facilita la creación de sitios web que se vean bien en diferentes dispositivos y navegadores.

[21/04/2024. Andrés Fernández Cano]
